# Online Payment Method Analysis

## Overview

This project provides a simple analysis of different online payment methods and their usage trends. With the rise of digital payments, it is important to understand which methods are most preferred by users. The project uses Python for data handling and visualization to demonstrate insights clearly.

---

## Objective

- To analyze transaction data across various online payment methods.  
- To visualize the data using charts for better understanding.  
- To identify the most and least popular payment methods.

---

## Tools and Libraries

- **Python 3** – programming language for analysis  
- **Pandas** – for creating and handling datasets efficiently  
- **Matplotlib** – for plotting charts and visualizations  

---

## Methodology

1. **Data Preparation** – Created a simple dataset with payment methods and number of transactions.  
2. **Analysis** – Used Pandas to calculate basic statistics and understand trends.  
3. **Visualization** –  
   - **Pie Chart**: Shows the proportion of each payment method in total transactions.  
   - **Bar Chart**: Compares the number of transactions across all methods.

---

## Key Findings

- **UPI** is the most popular payment method in this analysis, indicating its widespread use.  
- **Wallets** are the least used method.  
- Digital payments are mainly dominated by **UPI and cards**, highlighting user preference for convenience.

---

## Conclusion

This project demonstrates how Python can be used for simple data analysis and visualization. It helps beginners understand:

- Handling data with Pandas  
- Creating charts with Matplotlib  
- Interpreting trends from visualizations  

Overall, this project provides a clear and beginner-friendly insight into online payment preferences.
